# Color Picker — Extensión para Chrome

## Instalación

1. Descarga o clona este repositorio.
2. Abre Chrome y ve a `chrome://extensions`.
3. Activa el **Modo de desarrollador** (esquina superior derecha).
4. Haz clic en **Cargar extensión sin empaquetar**.
5. Selecciona la carpeta del proyecto.

## Uso

1. Haz clic en el ícono de la extensión en la barra de herramientas.
2. Presiona **Pick Color** para activar el selector de color.
3. Haz clic en cualquier punto de la pantalla para capturar el color.
4. El color se guarda automáticamente en el historial.
5. Haz clic en un color del historial para **copiar su código HEX** al portapapeles.
6. Usa el botón **Clear** para limpiar el historial.
